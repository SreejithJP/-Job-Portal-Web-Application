About This Application

Welcome to our Job Portal Application, a comprehensive platform designed to streamline the process of job searching and recruitment for both job seekers and job providers.

How It Works:

Job Seekers can:

  • Register on the platform by providing personal information, skills, and experience.
  • View available job listings with details like job title, company name, location, required skills, and experience.
  • Apply directly to listed jobs. The system prevents duplicate applications.
  • Track applied jobs, ensuring a clear view of application history.

Job Providers can:

  • Register their company and post job openings with detailed descriptions.
  • View applications submitted by job seekers for each listed position.
  • Manage applicants through an intuitive interface that lists all applicants for each job.

Technical Overview:

Frontend: Built using HTML, CSS, and JavaScript, providing an interactive and responsive user interface.

Backend: Powered by Spring Boot MVC, with RESTful APIs handling data exchange between the frontend and backend.

Database: Utilizes MongoDB to store user and job data, ensuring efficient data management and retrieval.

Key Features:

  • Secure Login System: Ensures only authenticated users can access job listings or post jobs.
  • Real-Time Updates: Buttons dynamically change from "Apply" to "Applied" without refreshing.
  • User-Centric Design: Easy navigation with a sidebar and clear status messages for actions.
